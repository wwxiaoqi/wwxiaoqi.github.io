Docker Compose 安装方法

安装

Docker Compose 近期推出了最新的V2版（V1是用Python编写的），但中文互联网众多教程还是使用V1的安装方式。

1
2
3
4
5

mkdir -p ~/.docker/cli-plugins/

curl -SL https://github.com/docker/compose/releases/download/v2.12.1/docker-compose-linux-x86_64 -o ~/.docker/cli-plugins/docker-compose

chmod +x ~/.docker/cli-plugins/docker-compose

以上命令mkdir新建了~/.docker/cli-plugins/文件夹，并在文件夹里面下载了最新的Docker Compose可执行文件，最后给该文件+x的可执行权限，然后就可以愉快的使用啦~

1

docker compose version

最新版本的执行方法是docker compose而不是docker-compose，注意中间的「-」以便区分。
