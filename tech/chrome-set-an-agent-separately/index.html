chrome 单独设置代理

先来说一下格式，比如说在谷歌浏览器下的 CMD 控制窗口下输入：

1
2

chrome.exe --proxy-server=127.0.0.1:10809
chrome.exe --proxy-server=192.168.241.74:10809

这样会启动 chrome 浏览器，并运行在本地代理的 8080 端口下，实际运用中可以将 127.0.0.1:8080 修改为指定的代理服务器，比如：

1
2
3
4
5

## http
https://xxxx.xx:443

## Socks
socks5://127.0.0.1:1080

并且配置可以在 Windows 快捷方式中添加，实现全 Chrome 设置代理。
