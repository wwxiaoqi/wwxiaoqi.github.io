在 Windows 上使用 mpv 播放 Dolby Vision 视频

接上文[1]，在 Windows 上使用 mpv 播放杜比视界(Dolby Vision)视频，你只需要：

  1. 右键 mpv.exe 应用程序在桌面创建快捷方式

  2. 右键这个快捷方式点开属性，在目标栏应用中添加

    --player-operation-mode=pseudo-gui --vo=gpu-next 参数

  3. 打开快捷方式

  4. 播放杜比视界(Dolby Vision)
