sudo poweroff         # 立刻关机
sudo shutdown -h now  # 立刻关机

sudo reboot           # 立即重启
sudo shutdown -r now  # 立即重启

sudo shutdown -h +5   # 五分钟后关机
sudo shutdown -r +5   # 五分钟后重启

init 6                # 利用 init 运行级别重启

shutdown +5 "系统将在5分钟后关闭" # 指定五分钟后关机，同时送出警告给登入用户
shutdown 的参数介绍
参数 介绍
-h 表示关机
-r 表示重启
now 表示立即执行
-f，-F 重新启动时不执行 fsck
-k 送出通知信息给所有用户，但不会实际关机
poweroff 和 shutdown 有什么区别？

shutdown 命令执行正常关机流程，包括清理工作和调用 halt 或 poweroff，而 poweroff 直接向硬件发送关机信号。在没有 ACPI[1] 的系统上，shutdown 只关闭系统，电源需手动关闭，而 poweroff 通过 ACPI 通知电源关闭。在有 ACPI 的系统上，两者无区别，系统会自动处理关机流程。

  1. ACPI（Advanced Configuration and Power Interface，高级配置和电源接口）是一种开放标准，它定义了操作系统和固件之间的接口，用于管理和控制硬件电源管理和配置问题，从而提高了计算机的电源效率和可移植性。
